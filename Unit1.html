Unit 1

1.1 What is the purpose of IT in modern business? Rapid facilitation of communication between workers, managers and customers is very important in todays world. Businesses also use IT to automate tasks in order to save time. Data can also be stored securely and analysed accurately and rapidly using IT

1.2 What resources and skills do you need to implement digital projects? Well, the right mix of software, hardware and technical knowledge is vital. You may need to have a tech guru on hand in case you get stuck. Persistance, curiosity and confidence are necessary for implementing most IT projects.

1.3 How do we plan and build a website? Firstly, we need to create the html files and put them in a folder. Then we need to create the html code for our web pages, including an internal navigation system. Next, we need to style the website using CSS adding backgrounds behind our text. Lastly we add the content between the html tags. Trello is a useful tool to aid in keeping track of the many tasks. You can see my trello by clicking this link. my trello page

1.4 What can cause IT projects to fail? Lack of communication between IT teams and businesses teams can cause IT projects to fail. Overambitious projects are also more likely to fail. Failure to anticipate possible problems you will encounter using the new equipment/software, for example incompatibilty problems may also lead to failure. Another thing that can go wrong is the incorrect choice of "off the shelf" software instead of "bespoke" software and vici versa.

1.5 How should we select the right technology for a given business? Firstly, we need to correctly ascertain what the business problems are we want the IT to fix. Alternatively, what processes and procedures could the IT make more efficient? Do we need bespoke or Off the shelf software? Is all the hardware necessary? do we really need powerful graphics cards for example

1.6How have Trello and Github help with building this website? Trello has helped by providing easliy accessible to do lists, which is the basis of any serious planning. Github provides a safe repository for code in case anything goes wrong with your computer. Also, it allows others to correct and view your code, so you are alone!

1.7 How does GDPR effect digital projects? GDPR is legislation designed to protect how EU citizens data is used and provides re restrictions on how data can be used and shared. As this website is not interactive no user data will be stored and GDPR will ill not be an issue. However websites that track users or store user input will need to make sure they abide by the legislation or face possible prosecution.
